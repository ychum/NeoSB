## 核心修改点

### 1. Tailwind配色系统 (tailwind.config.js)
- **背景变暗**: 从标准白色/黑色 → 深紫蓝/紫黑色系
- **主题色替换**: 蓝色/青绿 → 紫色/红色
- **新增魔剑色板**: 7种专属奇幻色彩
- **暗色模式优化**: 所有灰度改为紫色调渐进

### 2. Hero组件 (Hero.js)
- **渐变按钮**: 青绿渐变 → 紫红渐变
- **hover效果**: 蓝色阴影 → 紫色光晕
- **文本颜色**: 标准灰度 → 紫色调

### 3. 卡片组件 (InfoCard.js, BlogPostCard.js)
- **卡片边框**: 灰色边框 → 紫色边框
- **按钮样式**: 青绿色 → 紫红色
- **hover效果**: 蓝色提升 → 紫色提升

### 4. 文章头部 (PostHeader.js)
- **分类标签**: 蓝色背景 → 紫色背景
- **链接颜色**: 标准蓝 → 魔法紫

### 5. 评论组件 (HexoRecentComments.js)
- **链接颜色**: 标准蓝 → 紫红色系
